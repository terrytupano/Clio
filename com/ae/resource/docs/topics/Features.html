Introducción

Clio es una aplicación que le asiste en el proceso se calculo y relleno de las planillas para el pago de obligaciones fiscales.

Características

  • Las formas son editadas directamente dentro de una imagen digital de esta, permitiendo que el conocimiento previo que el usuario posee de las características de la planilla se asimile reduciendo el tiempo de aprendizaje.
  • Utiliza el concepto de contribuyente como única unidad básica de trabajo. De esta forma, una vez que se han introducido los datos dentro de la aplicación, ya se esta listo para comenzar a asignarle formas fiscales.
  • Los cálculos realizados en tiempo real permite una constante retroalimentación entre la aplicación y el usuario
  • Soporte para impresión normal o invertida: esto es debido a que la aplicación conoce donde se encuentra del pre-picado de cada una de las formas; Así, indica al usuario como debe colocar la forma física dentro de la impresora evitando que esta se dañe mientras se imprime.
  • La interfaz contable habilita al usuario cargar transacciones contables dentro de la aplicación. Luego de esto, la forma puede ser asignada a un contribuyente y luego impresa en 2 pasos permitiendo un alto grado de automatización.
  • El sistema de inventario permite que cada usuario adquiera solo las formas fiscales con la cuales trabaja permitiendo un uso justo de su aplicación.
