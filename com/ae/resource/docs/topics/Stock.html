Inventario

Según el diccionario, el inventario es una relación detallada de bienes o pertenencias. De forma similar, llamaremos inventario a la lista de formas que pueden ser asignadas o asociadas a un contribuyente en un momento dado. Dentro de la aplicación, el inventario de formas se presenta presionando el botón Inventario, el cual presenta el dialogo

La tabla que aparece en la parte inferior de la ventana le indica la situación actual de su inventario; indicando cuantas unidades de cada forma tiene a su disposición.

Para aclarar conceptos, imagine el inventario como una caja donde tiene guardadas sus planillas para las declaraciones fiscales. Cada vez usa una de estas, disminuye el contenido de estas en la caja y deben ser repuestas cada cierto tiempo comprando mas planillas en los agentes autorizados.

De manera análoga, cada vez que asocia una forma a un contribuyente, esta forma es descontada del inventario y debería ser repuesta para que haya existencia de dicha forma cuando se necesite nuevamente. Esta operación de reposición debe ser efectuada usando los componente contenidos en la pestaña ordenar

En la imagen puede verse que se desean 4 nuevas formas DPJ y se esta seleccionando una cantidad de DPN-R. Una vez satisfechos con la solicitud se presiona solicitar; el cual presentara un asistente que permitirá completar la operación.
Nota:
Este atento al contenido de esta ventana ya que si no tiene disponibilidad de formas, no podrá realizar declaraciones a los contribuyentes
