Contribuyente

Un contribuyente es la unidad principal de trabajo, a el se le carga movimiento contable y es a un contribuyente que se le asignan formas fiscales. Por lo tanto, debe tenerse cuidado al parametrizarlos ya que la aplicación constantemente esta consultando sus valores para la toma decisiones. Por Eje.: Si el contribuyente esta residenciado en el país, la aplicación no permite que se le asignen formas para no residentes.
Nota:
Si el Ud. Crea un contribuyente es porque esta consiente de que este, según la ley debe declarar o pagar impuestos. Los parámetros dados cuando se adiciona un nuevo contribuyente no están diseñados para determinar si esta persona (del tipo que sea) debe declarar o no.
